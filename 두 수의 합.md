## 문제 설명

---

<aside>
💡 정수 `num1`과 `num2`가 주어질 때, `num1`과 `num2`의 합을 return하도록 soltuion 함수를 완성해주세요.

</aside>

## 문제 풀이

---

1회차 → 통과

```jsx
function solution(num1, num2) {
    var answer = -1;
    answer = num1 + num2;
    return answer;
}
```

2회차 

```jsx
const solution = (num1, num2) => num1 + num2;
```
